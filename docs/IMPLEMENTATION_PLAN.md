## 3. 风险与缓解
- [x] 句子音频缺失或命名偏差 → 前缀匹配 + 缺失时跳过并提示（已汇总缺失列表，Snackbar 提示已实现）。

- [ ] 字体/排版差异 → 常见分辨率回归，必要时强制内置字体。

## 4. 验收标准（MVP）
- 冷启动 <1s，封面后进入阅读。
- 翻页流畅，进度保存与恢复正确。
- 朗读：点击后播放当前句子，能顺播当页句子；跨章节自动停止或重置。
- BGM 控制可用；翻页音可播放。
- 全程离线，无崩溃。
