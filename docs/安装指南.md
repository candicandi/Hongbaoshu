# Hongbaoshu 安装指南

## 为什么会出现安全警告?

当您从 GitHub 或其他非官方应用商店下载并安装 Hongbaoshu 时,可能会收到安全软件的警告提示。**这是正常现象**,主要原因包括:

1. **未经应用商店验证** - 应用未通过 Google Play 等官方渠道分发
2. **自签名证书** - 使用开发者自己的签名,而非商业代码签名证书
3. **安全软件的保守策略** - 对所有非官方渠道的应用都会发出警告

> **重要说明**: Hongbaoshu 是一个开源项目,所有源代码都可以在 [GitHub](https://github.com/ceasarXuu/Hongbaoshu) 上查看和审计。应用不包含任何恶意代码、广告或数据收集功能。

---

## 安装步骤

### 1. 下载 APK

从 [GitHub Releases](https://github.com/ceasarXuu/Hongbaoshu/releases) 页面下载最新版本的 `app-release.apk`。

### 2. 允许安装未知来源应用

根据您的 Android 版本,按照以下步骤操作:

#### Android 8.0 及以上版本

1. 点击下载的 APK 文件
2. 系统会提示"不允许安装未知应用"
3. 点击**设置**按钮
4. 开启**允许来自此来源的应用**开关
5. 返回并继续安装

#### Android 8.0 以下版本

1. 打开**设置** → **安全**
2. 开启**未知来源**选项
3. 点击下载的 APK 文件进行安装

### 3. 处理安全软件警告

如果您的设备安装了第三方安全软件(如 360 手机卫士、腾讯手机管家等),可能会弹出病毒警告:

#### 常见警告类型

- "检测到未知应用"
- "可能存在风险"
- "未经验证的应用"

#### 如何处理

1. **查看详细信息** - 点击警告查看具体原因
2. **选择信任** - 点击"仍要安装"、"继续安装"或"信任此应用"
3. **添加白名单** - 在安全软件中将 Hongbaoshu 添加到信任列表

> **提示**: 如果安全软件直接阻止安装,您可以:
> - 临时关闭安全软件的"应用安装监控"功能
> - 在安全软件设置中添加 Hongbaoshu 到白名单
> - 安装完成后再重新开启安全功能

## 验证应用完整性

为了确保您下载的是官方发布的应用,可以验证 APK 的签名指纹:

### 方法 1: 使用在线工具

1. 访问 [VirusTotal](https://www.virustotal.com/)
2. 上传您下载的 APK 文件
3. 查看扫描结果和签名信息
4. 对比 GitHub Release 页面提供的 SHA-256 指纹

### 方法 2: 使用命令行工具

如果您熟悉命令行,可以使用以下命令验证:

```bash
# 使用 apksigner (需要 Android SDK)
apksigner verify --print-certs app-release.apk

# 或使用 keytool
keytool -printcert -jarfile app-release.apk
```

### 官方签名信息

**APK 签名 SHA-256 指纹**:
```
(将在下次发布时更新)
```

如果您计算出的指纹与上述信息一致,说明应用未被篡改。

## 常见问题

### Q: 为什么不发布到 Google Play?

A: Hongbaoshu 是一个个人开源项目,主要用于技术学习和交流。由于项目包含的内容版权问题,不适合在 Google Play 等商业平台发布。

### Q: 应用会收集我的数据吗?

A: **不会**。Hongbaoshu 不包含任何数据收集、统计或广告功能。所有数据都存储在您的设备本地,不会上传到任何服务器。您可以查看源代码进行验证。

### Q: 安装后还会被安全软件拦截吗?

A: 首次安装后,将应用添加到安全软件的白名单,后续使用就不会再收到警告。

### Q: 如何卸载应用?

A: 与卸载其他应用相同:
1. 长按应用图标
2. 选择"卸载"或拖动到卸载区域
3. 确认卸载
