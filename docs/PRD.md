# 红宝书 App（毛主席语录）PRD v1.0

## 1. 产品定位

纯离线电子书产品，内置 1966 年版《毛主席语录》全文及朗读音频，专注阅读体验。
目标：简单、稳定、无网络、无登录、无复杂交互。

## 2. 核心功能概述

- 启动封面
- 正文阅读（含翻页动画 + 翻页音效）
- BGM 循环播放（可切歌/静音）
- 朗读模式（离线音频）
- 阅读进度自动记录
- 目录跳转
- 全部资源（文本、音频、BGM、封面图）随包内置，无需网络请求。

## 3. 启动与首页体验

### 3.1 启动流程

App 启动后展示红宝书封面（静态图 JPG/PNG）。
%
任意点击屏幕或向左滑动：

- 若首次使用 → 进入正文第一页
- 若非首次 → 自动跳转到上次阅读位置

### 3.2 封面设计要求

- 使用 1966 年版封面风格（红底+金字），不加入无关 UI 元素
- 无按钮、无菜单，保持“纸质书开篇感”

## 4. 正文阅读功能

### 4.1 文本内容

- 内置《毛主席语录》1966 年内容（章节与原排版段落结构）
- 字体：默认仿宋或苹方（iOS）/ Noto Serif SC（Android）
- 字号可固定，无需动态字号调整（保持简单）

### 4.2 仿真翻页动画

要求：

- 左右翻页
- 动画类似真实书页卷动效果（使用系统 GPU shader 或开源 PageCurl 动画库）
- 翻页手势：左右滑动、点击左右区域
- 翻页音效：每次翻页播放短音效（包内置 mp3 / wav）

### 4.3 阅读进度

- 自动持久化：记录当前章节 + 页码
- 使用本地存储（iOS UserDefaults / Android SharedPreferences）
- 无需手动书签

## 5. BGM 播放模块

### 5.1 BGM 设计

- 内置若干条背景音乐（红色经典伴奏或舒缓器乐）
- 循环播放
- App 内部无版权风险（确保自带音乐可自由使用或自行制作）

### 5.2 BGM 控制

在阅读页面右上角放置“音乐控制按钮”（不破坏简洁布局）：

- 点击一次：打开悬浮控制面板
- 播放/暂停
- 下一首
- 音量静音切换

### 5.3 状态持久化

- 记住用户是否开启/关闭音乐
- 记住当前音乐列表索引

## 6. 朗读模式（毛泽东音色）

### 6.1 内容要求

- 所有朗读音频提前离线制作好，随包内置
- 需覆盖全部章节（音频按章节切割）
- 格式：mp3 / aac，保证 app 体积可接受

### 6.2 朗读播放逻辑

阅读页面底部放置朗读按钮：

- 点击启动朗读模式：
  - 播放当前章节对应音频
  - 正文自动滚动到对应章节开头（不做逐句字幕联动，保持简单）
  - 离开该章节自动停止播放
- 暂停/继续按钮固定底部浮层显示

### 6.3 音频续播策略

- 不跨章节继续播放（保留简单逻辑）
- 切页时若仍在当前章节则继续播放

## 7. 目录（TOC）功能

### 7.1 进入方式

- 阅读页左上角按钮：⬅ 或 ☰（简洁图标）

### 7.2 内容

- 显示全部章节标题（按 1966 年版结构）
- 点击任意章节跳转到正文对应位置

### 7.3 展示形式

- 全屏列表
- 无搜索、无折叠层级（保持简单）

## 8. 本地资源结构（建议）

```
/assets
/text/mao_quotes_1966.json         # 正文以结构化 JSON 存储（章节数组）
/audio/bgm/*.mp3                   # BGM 列表
/audio/narration/chapter_XX.mp3    # 朗读音频
/images/cover.png                  # 启动画面封面
/sound/page_flip.wav               # 翻页音效
```

## 9. 系统架构（简化版）

### 9.1 模块划分

- Renderer（阅读器模块）：文本分页、翻页动画
- AudioManager（音频管理）：BGM + 翻页音效 + 朗读音频统一控制
- ProgressStore（本地持久化）：阅读进度、设置状态
- ContentLoader（内容加载）：加载本地 JSON 文本和本地音频资源
- UI 层：封面页、目录页、阅读页、朗读控制层

### 9.2 完全本地化

不包含以下能力：

- 登录
- 网络请求
- 云端同步
- 推送通知

## 10. 交互流程（核心）

### 10.1 初次使用

启动 → 封面页 → 点击 → 正文第一页 → 翻页浏览 → 目录 → 任意跳转

### 10.2 非首次使用

启动 → 封面页 → 点击 → 自动跳转上次阅读位置

## 11. 设备和兼容性

### 11.1 iOS

- iOS 14+
- SwiftUI 或 UIKit 均可实现（SwiftUI + UIViewRepresentable 处理翻页动画）

### 11.2 Android

- Android 8.0+
- Kotlin + Jetpack Compose（翻页动画可接入第三方 PageCurl View）

## 12. 非功能性要求

### 12.1 性能

- 启动封面加载 ≤ 1s
- 翻页动画流畅无明显卡顿（目标 60 fps）
- BGM 与朗读切换无爆音

### 12.2 安装包体积

- 控制在 50–150 MB（根据朗读音频数量）
- 尽可能压缩图片、优化音频码率（不影响可听度）

### 12.3 稳定性

- App 全程不崩溃、不卡死
- 不需要权限（除音频播放）
