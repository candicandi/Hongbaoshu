## 📖 项目简介

这是一个基于 **Jetpack Compose** 构建的现代化 Android 电子书阅读器应用。它不仅提供了优雅的阅读体验，还深度集成了 **"听书"** 功能，支持段落级的高亮跟随朗读。

### 核心功能

*   **沉浸式阅读**：精排版的文本显示，支持字体大小调节。
*   **同步朗读**：点击段落即可播放对应的真人朗读音频，阅读进度自动高亮跟随。
*   **背景白噪音**：内置多款背景白噪音（BGM），提供专注的阅读氛围。
*   **流畅交互**：基于 Compose 的现代化 UI，支持丝滑的翻页和手势操作。
*   **离线体验**：所有资源内置，无须网络连接即可完整体验。

## 🛠️ 技术栈

*   **语言**: Kotlin
*   **UI 框架**: Jetpack Compose (Material3)
*   **音频引擎**: AndroidX Media3 (ExoPlayer)
*   **架构**: MVVM
*   **构建工具**: Gradle (Kotlin DSL)

## 📱 如何使用

### 安装

您可以在本项目的 [Releases](../../releases) 页面下载最新的安装包 (`app-release.apk`) 进行体验。

### 编译运行

如果您是开发者，可以克隆本项目进行编译：

1.  克隆仓库：
    ```bash
    git clone [您的仓库地址]
    ```
2.  在 Android Studio 中打开项目。
3.  同步 Gradle 依赖。
4.  连接 Android 设备或模拟器，运行 `Run 'app'`。

### 操作指南

1.  **翻页**：左右滑动屏幕，或点击屏幕边缘区域进行翻页。
2.  **菜单**：双击屏幕顶部区域，或从顶部向下滑动，唤出功能菜单。
3.  **听书控制**：
    *   在菜单中开启/关闭“朗读模式”。
    *   点击正文中的任意段落，即可跳转播放该段落的朗读。
4.  **背景音**：在菜单中开启/关闭 BGM，调节音量或切换曲目。
